# AI-Powered Airline Catering Decision Support System

This project presents an AI-powered decision support system designed to improve airline catering operations. The system predicts meal requirements and estimates food waste percentages based on flight-related information, helping airlines optimize catering planning, reduce operational costs, and minimize food waste.

The prediction model was trained using Machine Learning techniques on a dataset containing 1,000 airline flight records. The trained model is deployed through the Hugging Face API and integrated into an intelligent conversational interface, enabling users to interact with the prediction system in a simple and efficient manner.
